# Sample Documents

This directory contains sample documents to help you test the HR Resume Analyzer application.

## Test the Application

1. Make sure your `.env` file points to this directory:
   ```
   FILE_PATH=./sample_documents
   ```

2. Run the application and update the vector store to index these sample documents

3. Try asking questions like:
   - "What skills are mentioned in the HR document?"
   - "What are the key policies for remote work?"
   - "Summarize the main points from these documents"

## Adding Your Own Documents

Replace these sample files with your own:
- Resume files (.pdf, .docx, .txt)
- Job descriptions
- HR policies
- Interview guides
- Salary benchmarks
